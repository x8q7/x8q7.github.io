JavaScript Object Notation, Javascript 对象表示法

注：JSON是一种数据格式，不是一种编程语言。

1.语法

JSON的语法可以表示，三种类型的值：

  • 简单值：字符串，数值，布尔值，null。不支持 undefined。
  • 对象：与JavaScript中对象相近，简单或者复杂键值对形式，但是，key 必须用双引号。
  • 数组：与JavaScript中数组相近，值或者对象的有序列表。

注：JSON不支持变量，函数，对象实例。
